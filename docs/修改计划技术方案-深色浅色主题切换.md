# 深色/浅色主题切换 技术方案

## 1. 背景与目标

**现状**：Web UI (`src/gateway/web-ui.ts`) 仅有深色主题，所有颜色均为硬编码的 hex 值，切换主题无任何机制。

**目标**：在不破坏现有深色主题外观的前提下，新增浅色主题支持，并提供便捷的切换开关。

---

## 2. 设计原则

1. **CSS 变量驱动** —— 所有颜色改用 CSS 变量定义，深色/浅色主题通过变量值切换实现
2. **零依赖** —— 纯原生实现，不引入任何外部库
3. **向后兼容** —— 默认保持深色主题，现有用户不受影响
4. **代码风格一致** —— 保持 `web-ui.ts` 现有的字符串拼接模板风格

---

## 3. 实现方案

### 3.1 CSS 变量体系设计

### 3.3 模板 CSS 元素迁移

将 `<style>` 块中所有硬编码颜色替换为 CSS 变量：

```css
/* 迁移前 */
body { background: #0d1117; color: #c9d1d9; }

/* 迁移后 */
body { background: var(--bg-primary); color: var(--text-primary); }
```

关键迁移点（第 59-202 行）：
- `body` 背景和文字色
- `.header` 背景、边框、文字色
- `.sidebar` 背景、边框
- `.message .bubble` 三种状态
- `.input-row textarea` / `button`
- `.md-content` 所有文字和背景色
- `.code-block .copy-btn`
- `.panel` / `.panel-header` / `.panel-body`
- `.setting-group` 表单元素
- `.tool-card`
- `.session-item` / `.session-item-delete`
- `::-webkit-scrollbar-thumb`

### 3.4 主题切换 UI

在 Header 右侧增加主题切换按钮：

```html
<!-- 在 .header-actions 中添加 -->
<button class="header-btn theme-toggle" onclick="toggleTheme()" title="Toggle theme">
  <span id="themeIcon">☀</span>
</button>
```

对应 JS 函数：

```javascript
function toggleTheme() {
  var html = document.documentElement;
  var current = html.getAttribute('data-theme');
  var next = current === 'light' ? 'dark' : 'light';
  html.setAttribute('data-theme', next);
  localStorage.setItem('theme', next);
  document.getElementById('themeIcon').textContent = next === 'light' ? '☀' : '🌙';
}
```

初始化时从 `localStorage` 读取保存的主题：

```javascript
(function initTheme() {
  var saved = localStorage.getItem('theme');
  if (saved) {
    document.documentElement.setAttribute('data-theme', saved);
    document.getElementById('themeIcon').textContent = saved === 'light' ? '☀' : '🌙';
  }
})();
```

## 4. 实现步骤

1. **提取颜色变量** —— 在 `<style>` 顶部插入 CSS 变量定义（`:root` + `[data-theme="light"]`）
2. **全局替换颜色** —— 将 `<style>` 块内所有 `#0d1117`/`#161b22` 等硬编码颜色替换为 `var(--xxx)` 变量引用
3. **添加切换按钮** —— 在 Header 的 `Settings` 按钮前插入主题切换按钮
4. **实现切换逻辑** —— 添加 `toggleTheme()` 和 `initTheme()` 函数
5. **本地存储持久化** —— 通过 `localStorage` 保存用户主题偏好
6. **验证深色主题** —— 确认迁移后深色主题外观完全一致
7. **验证浅色主题** —— 手动切换到浅色主题，检查各元素可读性

---

## 5. 注意事项

- **不修改 `RENDER_MARKDOWN_JS`** 中的硬编码颜色（如 `#8b949e` 用于代码块语言标签），该字符串内的颜色是 Markdown 渲染辅助信息，不属于主题系统
- **切换按钮样式**继承现有 `.header-btn` 类，无需新增 CSS
- **浅色主题滚动条** `--webkit-scrollbar-thumb` 背景色应适配为 `#d0d7de`
- **Session 列表的空状态文字** `#484f58` 在浅色主题下对比度需确保可读（建议浅色主题下用 `#8c959f`）
